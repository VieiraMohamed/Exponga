🌐 EXPONGA - Sitio Web Corporativo

Este repositorio contiene el código fuente del sitio web oficial de **EXPONGA**, una empresa especializada en la **organización y gestión de eventos de todo tipo**, desde celebraciones corporativas hasta eventos sociales y culturales.

🔗 Sitio web en línea: [www.exponga.com](https://www.exponga.com)

El sitio fue desarrollado con un enfoque moderno, responsivo y funcional, proporcionando una experiencia fluida tanto en dispositivos móviles como en escritorio. Además, cuenta con versiones en Español e Inglés para un alcance más amplio.

---

🛠️ Tecnologías Utilizadas

- **HTML5** – Estructura semántica del contenido
- **CSS3** – Diseño visual personalizado y responsivo
- **JavaScript** – Interactividad y mejoras en la experiencia del usuario
- **PHP** – Procesamiento de formularios y lógica del lado del servidor
- **Brevo (ex-Sendinblue)** – Integración de formularios de contacto y envío de correos automáticos
- **WordPress** – Blog corporativo autoadministrable y escalable

---

🌍 Soporte Multilingüe

El sitio ofrece contenido en dos idiomas:

- 🇪🇸 Español – Carpeta: `html/ES/`
- 🇬🇧 Inglés – Carpeta: `html/EN/`

Esto permite a EXPONGA conectar con una audiencia internacional sin perder identidad local.

---

📁 Estructura del Proyecto

```bash
EXPONGA/
├── css/            # Hojas de estilo (CSS)
├── img/            # Imágenes y recursos gráficos
├── php/            # Scripts PHP para formularios y lógica backend
├── html/
│   ├── ES/         # Versión en Español del sitio web
│   └── EN/         # Versión en Inglés del sitio web
├── index.html      # Página principal o redireccionamiento inicial
✉️ Formularios de Contacto
Los formularios están integrados con Brevo, permitiendo:

Recepción de mensajes en tiempo real

Confirmación automática al cliente

Gestión segura de datos

El procesamiento se realiza mediante scripts PHP ubicados en la carpeta php/, garantizando estabilidad y seguridad.

📰 Blog Corporativo
El blog fue implementado utilizando WordPress, permitiendo a EXPONGA publicar noticias, novedades y artículos relacionados con sus eventos de manera sencilla.
La plataforma es autoadministrable, adaptada al diseño general del sitio y completamente funcional desde dispositivos móviles.

🚀 Despliegue
El sitio puede desplegarse fácilmente en cualquier servidor que soporte PHP. Para un correcto funcionamiento, asegúrate de:

Colocar el contenido en la raíz del servidor

Tener PHP habilitado para los formularios

Configurar las rutas de Brevo correctamente

Instalar WordPress en un subdirectorio (ej. /blog) si se desea mantener el blog activo
